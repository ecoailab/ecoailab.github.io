EcoAI Lab은 스마트팩토리와 스마트홈의 에너지 데이터를 기반으로 수요 예측, 효율 개선, 운영 최적화 방법을 연구합니다.

주요 관심사는 재생에너지 예측, 에너지 사용 패턴 분석, 강화학습 기반 제어, 현장 적용 가능한 의사결정 시스템입니다.

## 문제

에너지 사용량은 설비 상태, 날씨, 사용자 행동, 생산 일정에 따라 크게 달라집니다. 단순한 평균값이나 규칙 기반 제어만으로는 변화에 빠르게 대응하기 어렵습니다.

## 접근 방법

- 시계열 데이터 기반 수요 예측
- 강화학습 기반 운영 정책 탐색
- 스마트팩토리·스마트홈 조건을 반영한 시뮬레이션
- 재생에너지 변동성을 고려한 의사결정 모델

## 기대 산출물

- 에너지 사용량 예측 모델
- 운영 최적화 알고리즘
- 실험용 데이터 처리 파이프라인
- 현장 적용 가능성을 검토한 연구 노트
