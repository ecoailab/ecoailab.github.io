EcoAI Lab은 IoT 센서 네트워크에서 수집되는 데이터를 기반으로 이상 탐지, 위협 분석, 안전한 지능형 서비스 구조를 연구합니다.

환경에너지 시스템과 연결된 센서 데이터의 신뢰성, 보안성, 운영 안정성을 높이는 것이 핵심 목표입니다.

## 문제

IoT 기반 에너지 시스템은 많은 센서와 네트워크에 의존합니다. 센서 오류, 데이터 변조, 네트워크 장애가 발생하면 AI 모델의 판단도 흔들릴 수 있습니다.

## 접근 방법

- 센서 데이터 품질 점검과 이상 탐지
- 물리보안·사이버보안 이벤트 분석
- 경량 AI 모델을 활용한 현장형 탐지
- 에너지 시스템 운영 조건을 반영한 보안 시나리오 설계

## 기대 산출물

- IoT 데이터 이상 탐지 모델
- 보안 이벤트 분석 절차
- 실험용 센서 데이터셋 구조
- 안전한 지능형 서비스 설계 지침
