EcoAI Lab은 환경에너지와 AI가 만나는 문제를 함께 다룰 파트너를 환영합니다. 협업은 작게 시작해도 좋습니다. 문제 정의, 데이터 검토, 세미나, 학생 프로젝트, 공동연구 과제처럼 상황에 맞는 방식으로 논의할 수 있습니다.

## 협업 가능 영역

- 스마트팩토리·스마트홈 에너지 데이터 분석
- IoT 센서 네트워크 설계와 보안 분석
- 강화학습 기반 운영 최적화
- 양자컴퓨팅과 AI 알고리즘 검토
- 멀티에이전트 기반 시뮬레이션

## 처음 연락할 때 알려주면 좋은 내용

- 해결하려는 문제
- 사용 가능한 데이터의 종류와 공개 가능 범위
- 기대하는 결과물
- 일정과 참여 가능한 인력
- 보안·개인정보·지식재산권 관련 제약
